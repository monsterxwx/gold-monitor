# 黄金猎手 - 实时金价与AI盯盘

一个基于Chrome/Edge扩展的实时金价监控插件，具有精美的UI、AI智能分析与预警功能。

## 功能特性

- 实时显示最新的金价走势
- 5秒自动零感知刷新
- 手动刷新功能
- 涨跌金额和涨跌幅分析
- 扩展图标徽章实时显示金价与涨跌趋势（白字红/绿底方案，降低视觉压迫）
- 简洁美观的弹出界面
- **[新增] 价格预警功能**：支持设置高/低预警价位，达标时弹出桌面通知提醒。
- **[新增] AI 分析功能**：集成 DeepSeek 大模型，结合实时金价提供交易建议及支撑/压力位分析。

## 使用说明

1. 安装后，浏览器工具栏会出现金价监控图标
2. 点击图标打开弹出窗口查看详细金价信息
3. 图标上的数字显示当前金价（整数部分）
4. 图标背景色：绿色表示上涨，红色表示下跌
5. **设置和AI功能**：
   - 点击“设置”按钮，配置您的 `DeepSeek API Key` 以及想要监控的价格区间（选填）。
   - 点击“AI 分析”按钮，系统将自动把最新的金价数据喂给AI，并返回给您操作建议。

## 本地安装与测试指南

如果您是开发者或正在调试本插件，请按照以下步骤加载：
1. 打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 并回车。
2. 开启右上角的 **开发者模式**。
3. 点击左上角的 **加载已解压的扩展程序**。
4. 选择本插件所在的文件夹（包含 `manifest.json` 的目录）。
5. 如果您修改了代码，请在 `chrome://extensions/` 页面点击该插件卡片上的 **刷新** 按钮，使代码修改生效。

## 关于 AI 分析逻辑的说明
本插件未让 DeepSeek 主动进行外部联网搜索。相反，由于插件本身已在持续获取高精度的实时金价数据，我们在您点击分析时，会将当前最新价格、涨跌额、涨跌幅、昨收价作为**强上下文**直接输入给 DeepSeek 模型。结合其实时传入的数据和内部固有的金融分析知识，能立刻得出适合当前的支撑位/压力位与加减仓判断，此方案安全、快速且精准。
